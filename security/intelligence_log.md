## Week 0 — 2026年4月30日（初回）

### 🟡 HIGH

**Anthropic Claude Mythos / Project Glasswing**
- AIが自律的にゼロデイ脆弱性を発見・エクスプロイト可能に
- 数十年間発見されなかったOpenBSDの脆弱性を発見した実績
- 危険すぎて一般公開せず、選ばれたセキュリティ専門家にのみ限定提供
- 当社への影響: Mythos級AIが攻撃者に使われれば、未知の脆弱性を大量に突かれる可能性
- 対策: 侵入前提設計を徹底。AI駆動のレッドチーミング体制を構築

**General Analysis — エージェントAIセキュリティ企業（$10M Seed）**
- 2026年4月に$10Mを調達。エージェントAI専用のセキュリティインフラを構築
- 当社にとって競合であり、技術の参考元でもある
- 対策: 彼らの公開技術・ブログ・採用動向を追跡

**ERC-8004 — 1ヶ月で45,000エージェント登録**
- ブロックチェーン上のエージェントID標準が急速に普及
- IDなりすまし攻撃の新パターンが出現する可能性
- 対策: ERC-8004の仕様を詳細調査。なりすまし対策を事前設計

- 米国カリフォルニアAI透明性法の施行動向
